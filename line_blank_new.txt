Методика расчетов2.1. Расчет пролета при отсутствии ветра и гололедаВес 1 метра провода без гололёда:Gп = m*9.8/1000 ,HВес 1 метра провода cable_name без гололёда:Gп(cable_name) = massa*9.8/1000 = cab_weight НИсходное натяжение провода определяется по формуле:H = (Gп*L^2/8*S)/1000, кНИсходное натяжение провода cable_name:Н(cable_name) = cab_weight*length^2/8*str_nach = start_h кН.Фактическая длина провода в пролете определяется по формуле:Lc = L+8*S^2/3*L, м.Фактическая длина провода cable_name:Lc(cable_name) = length+8*str_nach^2/3*length = factlen мДля упрощения последующих расчетов вводится промежуточная величина - длина провода без воздействия на него растягивающих нагрузок.Длина провода без воздействия на него растягивающих нагрузок определяется по формуле:Lu=Lc/(1+H/(A*E)), м.гдеА - поперечное сечение упругих элементов провода, мм2Е - модуль упругости, кН/мм2.Длина провода cable_name без воздействия на него растягивающих нагрузок:Lu(cable_name) = factlen/(1+start_h/(sech*modul)) = startlen м.Для учета влияния температуры рассчитывается длина провода без воздействия на него растягивающих нагрузок при температуре расчетного режима.Эта величина используется при расчете уравнения состояния провода для определения расчетной стрелы провеса.Luf=Lu(1+Tk(Tf-Tu)),гдеTf — температура расчетного режима;Tu — температура на момент измерений.Длина провода cable_name без воздействия на него растягивающих нагрузок при температуре +10:Luf(cable_name) = startlen*(1+tklr*10^-6*(10-temp)) = t_len м.Суммарная нагрузка на 1м провода равна нагрузке от собственного веса:Wf = G, НгдеG — вес 1 метра провода, для данного режима G=Gп.Суммарная нагрузка на 1м провода cable_name:Wf1(cable_name) = cab_weight НСтрела провеса при текущих условиях Sf определяется из кубического уравнения:Sf^3+aSf+b=0,гдеа = 3*(L^2-L*Luf)/8b = -3*Wf*L^3*Luf/(64*A*E)Дискриминант кубического уравнения вычисляется по формуле:D = (a/3)^3+(-b/2)^2при D большем или равным нулю:Sf={(-b/2)+[(a/3)^3+(-b/2)^2]^1/2}^1/3+{(-b/2)-[(a/3)^3+(-b/2)^2]^1/2}^1/3если D меньше нуля:Sf=2*((-a/3)^1/2)*cos{(1/3)*cos^-1[(-b/2)/(-a/3)^3/2]}Стрела провеса при текущих условиях для провода cable_name:a = a_norm;b = b_norm;D = d_norm;Стрела провеса расчетная,Sf = str_norm м;Зная стрелу провеса кабеля, определяем растягивающую нагрузку:Т(cable_name) = Wf*L^2/(8*Sf) ,кHРастягивающая нагрузка, действующая на провод cable_name:Т(cable_name) = nagr_norm кН.Аналогичным образом производится расчет стрел провеса и растягивающих нагрузок при других значениях температуры.2.2. Расчет пролета в режиме: Ветер отсутствует. Температура -5, максимальный гололед.Для учета влияния температуры рассчитывается длина провода без воздействия на него растягивающих нагрузок при температуре расчетного режима.Эта величина используется при расчете уравнения состояния провода для определения расчетной стрелы провеса.Luf=Lu(1+Tk(Tf-Tu)),гдеTf — температура расчетного режима;Tu — температура на момент измерений.Длина провода cable_name без воздействия на него растягивающих нагрузок при температуре -5:Luf(cable_name) = startlen*(1+tklr*10^-6*(-5-temp)) = calclen м.Нормативная гололедная нагрузка на 1 м провода Pгн, Н/м, определяется по формуле:Pгн = π*Ki*Kd*bэ*(d+Ki*Kd*bэ)*ρ *g*0.001, Н/мгдеKi – коэффициент, учитывающий изменение толщины стенки гололеда по высоте, принимаемый по  таблице 2.5.4 ПУЭ;Kd - коэффициент, учитывающий изменение толщины стенки гололеда в зависимости от диаметра провода, принимаемый по таблице  2.5.4 ПУЭ;bэ – толщина стенки гололеда, принимаемая в соответствии с 2.5.46 ПУЭ, мм;d – диаметр провода, мм;ρ – плотность льда, 0.9 г/см³;g – ускорение свободного падения, 9.8 м/с²;Нормативная гололедная нагрузка на 1 м провода cable_name:Ргн(cable_name) = 3.14*ki*kd*cg*(diam+ki*kd*cg)*0.9*9.8*0.001 = gol_norm Н.Расчетная линейная гололедная нагрузка на 1 м провода Ргр, Н/м, в соответствии с п.2.5.55 ПУЭ-7, определяется по формуле:Ргр = Ргн*ɣпг*ɣр*ɣf*ɣdгдеРнг – нормативная гололедная нагрузка;ɣпг – коэффициент надежности, принимаемый равным:1.0 – для ВЛ до 220 кВ;1.3 – для ВЛ 330 – 750 кВ;ɣр – региональный коэффициент, принимается от 1 до 1.5 на основании опыта эксплуатации;ɣf – коэффициент надежности по гололедной нагрузке, принимается равным 1,3 для 1 и 2 районов по гололеду; 1,6 – для 3 и выше районов по гололеду;ɣd – коэффициент условий работы равный 0.5.Расчетная линейная гололедная нагрузка на 1 м провода cable_name :Ргр(cable_name) =  gol_norm*1.0*1.0*kf*0.5 = golol Н.Суммарная нагрузка на 1м провода от веса провода, покрытого гололёдными отложениями определяется по формуле:Wf=Gп+Pгр, НгдеGп — вес 1 метра провода;Pгр – нормативная гололёдная нагрузка на 1 метр провода (см.п. 1.2);Суммарная расчетная нагрузка на 1м провода cable_name:Wf1(cable_name) = cab_weight+golol = wt_gol НСтрела провеса при текущих условиях Sf определяется из кубического уравнения:Sf^3+aSf+b=0,гдеа = 3*(L^2-L*Luf)/8b = -3*Wf*L^3*Luf/(64*A*E)Дискриминант кубического уравнения вычисляется по формуле:D = (a/3)^3+(-b/2)^2при D большем или равным нулю:Sf={(-b/2)+[(a/3)^3+(-b/2)^2]^1/2}^1/3+{(-b/2)-[(a/3)^3+(-b/2)^2]^1/2}^1/3Если D меньше нуля:Sf=2*((-a/3)^1/2)*cos{(1/3)*cos^-1[(-b/2)/(-a/3)^3/2]}Стрела провеса при текущих условиях для провода cable_name:a = a_gol;b = b_gol;D = d_gol;Стрела провеса расчетная,Sf = str_gol м;Зная стрелу провеса кабеля, определяем его нормативную растягивающую нагрузку:Т(вок) = Wf*L^2/(8*Sf), кHРасчетная растягивающая нагрузка, действующая на провод cable_name:Т(cable_name) = nagr_gol кН.2.3. Расчет пролета в режиме: температура -5, режим максимальной ветровой нагрузки по 2.5.54 ПУЭ-7. Гололед отсутствует.Нормативная ветровая нагрузка на провод в пролете Рwн, Н, действующая перпендикулярно проводу, в соответствии с 2.5.52 ПУЭ, определяется по формуле:Рwн = α*Kl*Kw*Cx*Wo*Fгдеαv – коэффициент, учитывающий неравномерность ветрового давления по пролету ВЛ, принимается равным:Ветровое давление, Па:   400         500         580 и болееКоэффициент αv:          0.76        0.71        0.7Kl – коэффициент, учитывающий влияние длины пролета на ветровую нагрузку, равный 1.2 при  длине пролета до 50 м, 1.1 – при 50 - 100 м, 1.05 – при 100 - 250 м, 1.0 – при 250 м и более;Kw – коэффициент, учитывающий изменение ветрового давления по высоте в зависимости от типа местности, определяемый по таблице 2.5.2 ПУЭ, в зависимости от ветрового района;Сх – коэффициент лобового сопротивления, принимаемый равным:1.1 – для проводов и тросов диаметром 20 мм и более;1.2 – для проводов и тросов диаметром менее 20 мм;Wo – нормативное ветровое давление, Па, принимаемое по таблице 2.5.1 ПУЭ, в зависимости от ветрового района.F – площадь прямоугольного диаметрального сечения провода, м², определяемая по формуле:F = d*L*0,001где d – диаметр провода, мм;L – длина пролета, м.Нормативная ветровая нагрузка на провод cable_name в пролете:Pwн(cable_name) = av*kl*kw*cx*vd*length*diam*0.001 = vet_norm Н;Расчетная ветровая нагрузка на провод, Pw, Н, в соответствии с п.2.5.54 ПУЭ-7, определяется по формуле:                Pw = Pwн* ɣнw*ɣр*ɣfгдеPwн – нормативная ветровая нагрузка;ɣнw,ɣр  - принимается согласно 2.5.54 ПУЭ-7ɣf – коэффициент надежности по ветровой нагрузке, равный 1.1:Расчетная ветровая нагрузка на провод cable_name в пролете:Pw(cable_name) = vet_norm*1.0*1.0*1.1 = vetr Н;Расчетная ветровая нагрузка на 1 м провода Pw0 = Pw / L, НРасчетная ветровая нагрузка на 1 м провода cable_name:Pw0(cable_name) = vet_metr HСуммарная нагрузка на 1м провода от веса провода (вертикальная компонента) и давления ветра (горизонтальная компонента) определяется по формуле:	 Wf = (G^2+Pwо^2)^1/2, НгдеG — вес 1 метра провода, для данного режима G=Gп;Суммарная нагрузка на 1м провода cable_name:Wf(cable_name) = (cab_weight^2+vet_metr^2)^1/2 = wvet_r НСтрела провеса при текущих условиях Sf определяется из кубического уравнения:Sf^3+aSf+b=0,гдеа = 3*(L^2-L*Luf)/8b = -3*Wf*L^3*Luf/(64*A*E)Дискриминант кубического уравнения вычисляется по формуле:D = (a/3)^3+(-b/2)^2при D большем или равным нулю:Sf={(-b/2)+[(a/3)^3+(-b/2)^2]^1/2}^1/3+{(-b/2)-[(a/3)^3+(-b/2)^2]^1/2}^1/3если D меньше нуля:Sf=2*((-a/3)^1/2)*cos{(1/3)*cos^-1[(-b/2)/(-a/3)^3/2]}Стрела провеса при текущих условиях для провода cable_name:a = a_vet;b = b_vet;D = d_vet;Стрела провеса расчетная,Sf = str_vet м;Зная стрелу провеса кабеля, определяем растягивающую нагрузку:Т(cable_name) = Wf*L^2/(8*Sf) ,кHРасчетная растягивающая нагрузка, действующая на провод cable_name:Т(cable_name) = nagr_vet кН.2.4. Расчет пролета в режиме: температура -5, максимальный гололед, ветровое давление 0,25 от максимального.Нормативная ветровая нагрузка на провод в пролете при гололеде Pwгн, Н, согласно 2.5.52 ПУЭ определяется по формуле:Pwгн = αwg*Kl*Kw*Cx*Wг*Fгдеαwg – коэффициент, учитывающий неравномерность ветрового давления по пролету ВЛ:Ветровое давление, Па:до 200   240      280       300       320       360       400Коэффициент αwg:          1    0.94     0.88      0.85      0.83      0.8       0.76Kl – коэффициент, учитывающий влияние длины пролета на ветровую нагрузку, равный 1.2 при  длине пролета до 50 м, 1.1 – при 500 -
