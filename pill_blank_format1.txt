Методика расчетов

Нагрузка от собственного веса проводов
Нагрузка, передаваемая на опору под действием собственного веса проводов и кабелей, определяется по формуле:
Gп = (m*9.8*Lправ + m*9.8*Lлев)/2 ,Н
где 
m пр. – масса провода, кг/км;
9.8 – ускорение свободного падения;
L – длина пролета, м.


Нагрузка от собственного веса провода xxxx:
Gп(xxxx) = ({val.wCabLeft}+{val.wCabRight})/2 = val.g1_1 Н.


 Гололедная нагрузка 
Нормативная гололедная нагрузка на 1 м провода Pгн, Н/м, определяется по формуле:

Pгн = π*Ki*Kd*bэ*(d+Ki*Kd*bэ)*ρ *g*0.001, Н/м
где 
Ki – коэффициент, учитывающий изменение толщины стенки гололеда по высоте, принимаемый по  таблице 2.5.4 ПУЭ;
Kd - коэффициент, учитывающий изменение толщины стенки гололеда в зависимости от диаметра провода, принимаемый по таблице  2.5.4 ПУЭ;
bэ – толщина стенки гололеда, принимаемая в соответствии с 2.5.46 ПУЭ, мм;
d – диаметр провода, мм;
ρ – плотность льда, 0.9 г/см³;
g – ускорение свободного падения, 9.8 м/с²;

Нормативная гололедная нагрузка на 1 м провода xxxx:
Ргн(xxxx) = 3,14*{val.ki}*{val.kd}*{val.cg}*({val.diam}+{val.ki}*{val.kd}*{val.cg})*0.9*9.8*0.001 = {val.wGolNorm} Н. 

Расчетная линейная гололедная нагрузка на 1 м провода Ргр, Н/м, воспринимаемая опорами, определяется по формуле:

Ргр = Ргн*ɣпг*ɣр*ɣf*ɣd
где
Рнг – нормативная гололедная нагрузка;
ɣпг – коэффициент надежности, принимаемый равным:
          1.0 – для ВЛ до 220 кВ;
          1.3 – для ВЛ 330 – 750 кВ;
ɣр – региональный коэффициент, принимается от 1 до 1,5 на основании опыта эксплуатации;
ɣf – коэффициент надежности по гололедной нагрузке при расчете по первой и второй группам предельных состояний, принимается равным 1,3 для 1 и 2 районов по гололеду; 1,6 – для 3 и выше районов по гололеду;
ɣd – коэффициент условий работы равный:
         1.0 – при расчете по первой группе предельных состояний;
         0.5 – при расчете по второй группе предельных состояний.

Расчетная линейная гололедная нагрузка на 1 м провода xxxx, воспринимаемая опорами:
При расчете по первой группе предельных состояний
Р1гр(xxxx) = {val.wGolNorm}*1.0*1.0*{val.kf}*1 = {val.wGolol_1} Н; 
При расчете по второй группе предельных состояний
Р2гр(xxxx) =  {val.wGolNorm}*1.0*1.0*{val.kf}*0,5 = {val.wGolol_2} Н. 

Суммарная расчетная нагрузка, передаваемая на опору проводом, покрытым гололедом,  определяется по формуле:

Gпг = Gп + (Ргр*Lправ. + Pгр*Lлев.)/2
где 
Ргр – расчетная линейная гололедная нагрузка на 1 метр провода;
Gп — нагрузка от собственного веса провода;
L – длина пролета, м.
  
Нагрузка от веса провода xxxx, покрытого гололедом, воспринимаемая опорой:
При расчете по первой группе предельных состояний 
G1пг(xxxx) = val.g1_1 + ({val.wGolol_1}*{val.len_left} + {val.wGolol_1}*{val.len_right})/2 = {val.g1_2} Н;
При расчете по второй группе предельных состояний 
G2пг(xxxx) = val.g1_1 + ({val.wGolol_2}*{val.len_left}+ {val.wGolol_2}*{val.len_right})/2 = {val.g2_2} Н.


 Ветровая нагрузка.
    Нормативная ветровая нагрузка на провод Рwн, Н, действующая перпендикулярно проводу, в соответствии с 2.5.52 ПУЭ, определяется по формуле:

Рwн = α*Kl*Kw*Cx*Wo*F

где
α – коэффициент, учитывающий неравномерность ветрового давления по пролету ВЛ, принимается равным:
Ветровое давление, Па:   400         500         580 и более
Коэффициент αv:          0.76        0.71        0.7 
Kl – коэффициент, учитывающий влияние длины пролета на ветровую нагрузку, равный 1.2 при  длине пролета до 50 м, 1.1 – при 50 - 100 м, 1.05 – при 100 - 250 м, 1.0 – при 250 м и более;
Kw – коэффициент, учитывающий изменение ветрового давления по высоте в зависимости от типа местности, определяемый по таблице 2.5.2 ПУЭ, в зависимости от ветрового района;
Сх – коэффициент лобового сопротивления, принимаемый равным:
        1.1 – для проводов и тросов диаметром 20 мм и более;
        1.2 – для проводов и тросов диаметром менее 20 мм;
Wo – нормативное ветровое давление, Па, принимаемое по таблице 2.5.1 ПУЭ, в зависимости от ветрового района.
F – площадь прямоугольного диаметрального сечения провода, м², определяемая по формуле:

                          F = d*L*0,001
                          где d – диаметр провода, мм;
                          L – длина пролета, м.

Нормативная ветровая нагрузка на провод xxxx:
Pwн(xxxx)прав = {val.av}*{val.klRight}*{val.kw}*{val.cx}*#vd*{val.len_right}*{val.diam}*0.001 = {val.wLNormR} Н;

где
Pwн – нормативная ветровая нагрузка;
ɣнw,ɣр  - принимается согласно 2.5.54 ПУЭ
ɣf – коэффициент надежности по ветровой нагрузке на конструкции опоры, равный:
       1.3 – при расчете по первой группе предельных состояний;
       1.1 – при расчете по второй группе предельных состояний.

Тяжения в проводах 
Расчет стрел провеса и тяжения провода в пролете ВЛ выполняется решением уравнения состояния провода. 
 Тяжения в проводах при ветровой нагрузке (I Н.Р.)
Провод xxxx
Исходные данные, необходимые для расчета: 
Температура  -5˚С.
Длина пролета, L, м:
Lлев = {val.len_left}
Lправ = {val.len_right}
Стрела провеса начальная, S, м:  
Sлев = #strLeftSupp
Sправ = {val.str_right}
Масса провода, m, кг/км  
m = {val.massa}
Диаметр провода, D, мм:
D = {val.diam}
Поперечное сечение провода, А, мм2:
А = {val.sech}
Модуль упругости, E, кН/мм²:
Е = {val.modul}
Температурный коэффициент линейного расширения, Тк, 1/С:
Тk = {val.tklr}*10^-6

Вес 1 метра провода без гололёда:

	Gп = m*9.8/1000 ,H

Вес 1 метра провода xxxx без гололёда:
Gп(xxxx) = {val.massa}*9.8/1000 = {val.wCab} Н/м

Исходное натяжение провода определяется по формуле: 
	H = (Gп*L^2/8*S)/1000, кН

Исходное натяжение провода xxxx:
Нлев(xxxx) = {val.wCab}*{val.len_left}^2/8*#strLeftSupp = #startH_left кН.
Нправ(xxxx) = {val.wCab}*{val.len_right}^2/8*{val.str_right} = #startH_right кН.

Фактическая длина провода определяется по формуле:
	Lc = L+8*S^2/3*L, м.

Фактическая длина провода xxxx:
Lc(xxxx)лев = {val.len_left}+8*#strLeftSupp^2/3*{val.len_left} = #fact_length_left м
Lc(xxxx)прав = {val.len_right}+8*{val.str_right}^2/3*{val.len_right} = #fact_length_right м

Для дальнейших расчетов необходимо знать длину провода между опорами без действующих на него растягивающих нагрузок. 
Данная величина называется длиной провода в ненагруженном состоянии. 
Исходную длину ненатянутого провода определяем по формуле: 

	 Lu=Lc/(1+H/(A*E)), м.
где
А - поперечное сечение кабеля, мм2 
Е - модуль упругости, кН/мм2.

Исходная длина ненатянутого провода xxxx:

Стрела провеса при текущих условиях Sf определяется из кубического уравнения: 
	Sf3+aSf+b=0, 
где
а = 3*(L^2-L*Luf)/8
b = -3*Wf*L^3*Luf/(64*A*E)

Дискриминант кубического уравнения вычисляется по формуле: 
	D = (a/3)^3+(-b/2)^2
при D>=0

Sf={(-b/2)+[(a/3)^3+(-b/2)^2]^1/2}^1/3+{(-b/2)-[(a/3)^3+(-b/2)^2]^1/2}^1/3
при D<0
Sf=2*((-a/3)^1/2)*cos{(1/3)*cos^-1[(-b/2)/(-a/3)^3/2]}
